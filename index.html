Sobre mim

Meu nome é Kauan Leandro, tenho 20 anos e mexo com computadores desde criança. Sempre achei a programação muito fascinante, apesar disso, nunca a estudei. Comecei a estudar programação no final de 2024, quando conheci o curso Full-Stack da Danki-code. Por enquanto, estou focado na área de Front-end.

Sempre achei, e ainda acho, muito interessante que, com alguns caracteres, podemos criar um site ou um aplicativo do zero.

Projetos

*passe o mouse nos projetos*

Landing page básica

Modelo de landing page desenvolvido como exercício do curso Danki Code. O foco foi na construção de uma página com layout responsivo utilizando apenas HTML e CSS, explorando conceitos como flexbox, estruturação semântica e design limpo.

YouTube

Aqui tentei recriar o YouTube usando somente HTML e CSS.

Formulário Funcional

Permite ao usuário enviar mensagens diretamente para um e-mail configurável dinamicamente via JavaScript. O formulário utiliza o serviço formsubmit.co para enviar os dados sem a necessidade de backend próprio.

Calculadora Simples

Calculadora básica para operações matemáticas simples. Esse projeto reforçou meu entendimento sobre variáveis, estruturas condicionais (if/else/else if) e lógica de programação em JavaScript. Foi essencial para consolidar fundamentos importantes da linguagem.

Pedra, Papel ou Tesoura

Um jogo clássico desenvolvido com o objetivo de praticar JavaScript. Aprendi a lidar com eventos de clique, a criar um sistema de decisão lógica e a melhorar a interação do usuário com a interface.

Lista de Supermercado

Uma lista de compras que utiliza o localStorage para salvar os dados localmente no navegador. Esse projeto foi fundamental para eu aprender como trabalhar com arrays, manipular o DOM com innerHTML e utilizar JSON para armazenar dados.

Music Player

Este é um projeto de player de música interativo desenvolvido com*HTML, CSS e JavaScript puro, com o objetivo de praticar conceitos como manipulação do DOM, eventos, localStorage, e lógica de programação.

Lista de Tarefas

Um projeto simples de lista de tarefas criado com HTML, CSS e JavaScript, oferecendo modo escuro e armazenamento local. Inicialmente desenvolvido para praticar JavaScript, acabou se tornando uma grande oportunidade de aprendizado.

Esses não são todos os meus projetos, são só os que eu mais gostei.
